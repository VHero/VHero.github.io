Github+Hexo搭建博客教程(三)

将Github账户和Hexo博客关联，以及如何发布文章，将文章同步到Github上面。

Github账户和Hexo关联

注意
1.因yml格式问题,所有键值对的”:”冒号后面必须跟有一个空格
2.自Hexo 3.0以后,type类型都为 git,而非 github

然后在Git Bash下面输入以下命令，即可完成部署和同步。

有的电脑在执行hexo d命令时,会出现ERROR Deployer not found: git 的错误提示,该错误原因有以下几点

1.在_config.yml文件中,deploy模块中冒号后面没有加空格
2.在该文件夹下执行 npm install hexo-deployer-git –save 命令后重新执行 hexo d命令

到此，Github账户和Hexo博客关联就到此结束。

注意
1.每次在本地进行改动后,不管是什么改动,都需要在D:\Hexo 文件夹下打开Git Bash,执行hexo g
2.如果需要同步到github中,则在执行完hexo g命令后,再次执行hexo d即可
3.在执行hexo d命令时,会让输入你的github邮箱和密码,在输入密码时为了安全考虑,输入字符时是不会显示的,请放心输入
