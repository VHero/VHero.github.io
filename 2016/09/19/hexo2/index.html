Github+Hexo搭建博客教程(二)

Github Page的创建，以及主题的选择和Github Page的发布。

注册Github

在此处进行注册,操作比较简单。

创建Repository

在自己的github的主页点击那个绿色的New Repository创建Repository

进入创建页面后必须保证仓库的名字与用户名一致，比如我的用户名为vhero，那么仓库名为vhero.github.io，点击Create repository，进入该仓库，点击右边的Settings，往下面拉找到Github Pages，


点击 Launch automatic page generator ，让Github生成Github Page。

进入下一页，点击绿色按钮 Continue to layouts 进入下一页。

这个界面是选择主题，选择完毕，点击绿色的按钮Publish page


然后就能进入这个页面，在浏览器输入VHero.github.io这个域名就能进入自己的博客。
